# SASPD923L

Terminal Portfolio — un portfolio interactif en ligne de commande.

## Concept

Un portfolio qui simule un terminal Unix. Les visiteurs tapent des commandes pour explorer les projets, compétences et informations.

## Commandes disponibles

- `help` — Liste toutes les commandes
- `about` — À propos
- `projects` — Projets réalisés
- `skills` — Compétences techniques
- `contact` — Informations de contact
- `clear` — Effacer le terminal
- `banner` — Afficher la bannière

## Technologies

- HTML5, CSS3, JavaScript vanilla
- Aucun framework, aucune dépendance
